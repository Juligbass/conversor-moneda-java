#  Conversor de Monedas en Java

Proyecto desarrollado como parte del **Challenge de Alura Latam**.

##  Descripción

Este proyecto es un conversor de monedas que consume una API de tasas de cambio para obtener valores actualizados entre diferentes divisas.

El programa permite al usuario seleccionar diferentes conversiones desde la consola.

## Tecnologías utilizadas

* Java 17
* HttpClient
* Gson
* ExchangeRate API

## Funcionalidades

* Consulta tasas de cambio en tiempo real
* Conversión entre monedas
* Menú interactivo en consola
* Consumo de API externa

## Estructura del proyecto

```
src/
 └── Main.java
lib/
 └── gson.jar
```

##  Cómo ejecutar

1. Clonar el repositorio
2. Abrir el proyecto en IntelliJ
3. Ejecutar la clase `Main`
